### 문제 설명

<p>루트 없는 트리가 주어진다. 이때, 트리의 루트를 1이라고 정했을 때, 각 노드의 부모를 구하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에 노드의 개수 N (2 ≤ N ≤ 100,000)이 주어진다. 둘째 줄부터 N-1개의 줄에 트리 상에서 연결된 두 정점이 주어진다.</p>

### 출력 

 <p>첫째 줄부터 N-1개의 줄에 각 노드의 부모 노드 번호를 2번 노드부터 순서대로 출력한다.</p>

### 풀이
<pre>문제 설명: 연결된 트리들을 그려보면 아래와 같은데, 이때 1부터 7까지 각각의 부모 노드가 누구인지 출력하면 된다. 
    1
  4   6
2  7    3
          5

방문체크할 배열
부모노드를 담을 배열
인접리스트
이 필요하다. 
 
DFS
1번 노드에서 시작한다.
현재 노드(now)의 인접 리스트를 확인하면서, 연결된 노드(next)가 아직 방문되지 않았다면:
parent[next] = now (부모 기록)
방문 체크
DFS(next) 재귀 호출
이렇게 해서 모든 노드를 탐색하며 각 노드의 부모를 기록한다

BFS
dfs와 비슷하지만, 
큐에서 노드를 하나 꺼내(now), 그 노드와 연결된 모든 인접 노드(next)를 확인한다.
연결된 노드가 아직 방문되지 않았다면 
parent[next] = now (부모 기록)
방문 체크
큐에 대입한다.
