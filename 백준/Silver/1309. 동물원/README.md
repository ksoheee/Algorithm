### 문제 설명

<p>어떤 동물원에 가로로 두칸 세로로 N칸인 아래와 같은 우리가 있다.</p>

<p><img alt="" src="https://www.acmicpc.net/upload/201004/dnfl.JPG" style="height:223px; width:224px"></p>

<p>이 동물원에는 사자들이 살고 있는데 사자들을 우리에 가둘 때, 가로로도 세로로도 붙어 있게 배치할 수는 없다. 이 동물원 조련사는 사자들의 배치 문제 때문에 골머리를 앓고 있다.</p>

<p>동물원 조련사의 머리가 아프지 않도록 우리가 2*N 배열에 사자를 배치하는 경우의 수가 몇 가지인지를 알아내는 프로그램을 작성해 주도록 하자. 사자를 한 마리도 배치하지 않는 경우도 하나의 경우의 수로 친다고 가정한다.</p>

### 입력 

 <p>첫째 줄에 우리의 크기 N(1≤N≤100,000)이 주어진다.</p>

### 출력 

 <p>첫째 줄에 사자를 배치하는 경우의 수를 9901로 나눈 나머지를 출력하여라.</p>


 ### 풀이
 <pre>문제 이해 
각 행에 사자 0마리 또는 왼쪽 1마리, 오른쪽 1마리만 배치 가능
세로로 붙은 같은 열에 연속해서 배치 할 수 없음

상태 정의
dp[i][j] : i번째 줄까지 다 배치했을 때 경우의 수
j = 0 : i번째 줄에 사자 없음
j = 1 : i번째 줄 왼쪽에만 사자
j = 2 : i번째 줄 오른쪽에만 사자

초기값
dp[1][0] = 1 (첫 줄에 사자 없음)
dp[1][1] = 1 (첫 줄 왼쪽)
dp[1][2] = 1 (첫 줄 오른쪽)

점화식
dp[i][0]= dp[i-1][0]+dp[i-1][1]+dp[i-1][2] : 이번줄에 아무것도 두지 않으면 이전 줄은 어떤 상태든 가능
dp[i][1]= dp[i-1][0]+dp[i-1][2] : 이번 줄 왼쪽에 두면 이전 줄은 아무것도 없거나 오른쪽만 가능
dp[i][2]= dp[i-1][0]+dp[i-1][1] : 이번 줄 오른쪽에 두면 이전 줄은 아무것도 없거나 왼쪽만 가능 

예를 들어 N=2
첫째 줄 / 둘째 줄 가능한 배치:
(0, 0)
(0, L)
(0, R)
(L, 0)
(L, L)  x (왼쪽 세로 연속)
(L, R) o (첫째 줄은 왼쪽, 둘째 줄은 오른쪽 → 세로 겹침 없음)
(R, 0)
(R, L) o (첫째 줄 오른쪽, 둘째 줄 왼쪽 → 세로 겹침 없음)
(R, R) x (오른쪽 세로 연속)
dp[2][0] = dp[1][0]+dp[1][1]+dp[1][2] = 1+1+1 = 3
dp[2][1] = dp[1][0]+dp[1][2] = 1+1 = 2
dp[2][2] = dp[1][0]+dp[1][1] = 1+1 = 2

최종값 : (dp[N][0]+dp[N][1]+dp[N][2]) % 9901
 </pre>
