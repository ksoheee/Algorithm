### 문제 설명

<p>자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.</p>

<ul>
	<li>1부터 N까지 자연수 중에서 중복 없이 M개를 고른 수열</li>
	<li>고른 수열은 오름차순이어야 한다.</li>
</ul>

### 입력 

 <p>첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 8)</p>

### 출력 

 <p>한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.</p>

<p>수열은 사전 순으로 증가하는 순서로 출력해야 한다.</p>

### 풀이
<p>중복 없는 조합을 구하는 문제이다. </p>
<p>오름차순이면서 M의 길이까지 나열 가능한 수열이여야 한다.
그 이유는 오름차순 조건이 없으면 중복된 조합이 여러번 나오기때문</p>
<p>depth와 현재위치, 즉 어디서부터 시작하는지를 의미하는 변수 at을 추가한다.</p>
<p>i는 at부터 탐색하도록 한다. 현재 깊이를 index로 하여 해당위치에 i값을 담는다. 현재 i 값보다 다음 재귀에서 더 커야 하므로 +1, 깊이 또한 1증가 시켜 재귀를 호출한다.</p>
<p>지금 재귀에 들어가기 전에 그 전 수보다 큰 수에서만 탐색하도록 하고 있기 때문에 중복방지는 따로 할 필요가 없다.</p>
