### 문제 설명

<p>돌 게임은 두 명이서 즐기는 재밌는 게임이다.</p>

<p>탁자 위에 돌 N개가 있다. 상근이와 창영이는 턴을 번갈아가면서 돌을 가져가며, 돌은 1개 또는 3개 가져갈 수 있다. 마지막 돌을 가져가는 사람이 게임을 이기게 된다.</p>

<p>두 사람이 완벽하게 게임을 했을 때, 이기는 사람을 구하는 프로그램을 작성하시오. 게임은 상근이가 먼저 시작한다.</p>

### 입력 

 <p>첫째 줄에 N이 주어진다. (1 ≤ N ≤ 1000)</p>

### 출력 

 <p>상근이가 게임을 이기면 SK를, 창영이가 게임을 이기면 CY을 출력한다.</p>

### 풀이
<pre>풀이 1
짝수일때 상근이가 이기고, 홀수 일때 찬영이가 이긴다.

풀이 2 - 승패 여부 기록
Dp 상태: 돌이 i 개 있을 때 현재 차례인 사람이 이길수 있는지 여부 (상근이가 이기면 true)

점화식
“내가 한 수 둔 뒤 상대가 진다면 나는 이길 수 있다”
!dp[i-1] : 내가 1개 가져갔을 때 상대방이 지는 경우
!dp[i-3] : 내가 3개 가져갔을 때 상대방이 지는 경우
둘 중 하나라도 된다면 내가 승리 

예시
dp[4] = !dp[3] || !dp[1]
       = !true || !true
       = false || false
       = false
=> 돌이 4개일때 현재 턴인사람이 반드시 진다.

풀이 3 - 최소 횟수/최소 비용 구하기 
dp[i] = 돌이 i 개 있을 때, 최소 몇번의 턴만에 게임이 끝나는지

돌이 i개 남았을 때
1개 가져가기 -> 남은 돌개수는 i-1 ->dp[i-1]번의 추가 턴 필요
3개 가져가기 -> 남은 돌개수는 i-3 ->dp[i-3]번의 추가 턴 필요
둘중에 더 빠르게 끝나는 쪽(더 작은 값)을 선택해야 한다.

그리고 지금 돌을 가져가는 행동(1개 또는 3개)를 하고 나면 한 턴이 추가 되므로 +1을 해준다.</pre>
